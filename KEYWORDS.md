# نظام الفلترة - كلمات مفتاحية إيران والتصعيد العسكري

## نظرة عامة

يستخدم النظام نظام فلترة ثلاثي الطبقات للتقاط أخبار إيران والتصعيد العسكري بالعبرية:

### الطبقة الأولى (أساسية - يجب وجود واحدة)
كلمات أساسية تشير مباشرة إلى إيران:
- **עברית**: איראן, טהראן, IRGC, חומיני, המנהיג העליון
- **English**: Iran, Tehran, IRGC, Khamenei, Supreme Leader
- **العربية**: إيران, طهران, الحرس الثوري, خامنئي, المرشد الأعلى

### الطبقة الثانية (أسماء العمليات)
تلتقط أخبار العمليات العسكرية حتى بدون ذكر إيران:
- **עברית**: שאגת האריה, מגן יהודה, ההבטחה הנכונה, פתח חיבר
- **English**: Lion's Roar, Shield of Judah, True Promise, Fateh Khyber
- **العربية**: زئير الأسد, درع يهودا, الوعد الصادق, فاتح خيبر

### الطبقة الثالثة (سياق موسّع)
كلمات سياق عام مرتبطة بالتصعيد:
- **شخصيات**: נתניהו (Netanyahu), טראמפ (Trump), ג'בארי (Jabbari)
- **تحالفات**: ציר ההתנגדות (Axis of Resistance), חזבאללה (Hezbollah)
- **مواقع نووية**: נטנז (Natanz), פורדו (Fordow), אספהן (Isfahan)
- **أسلحة**: טילים בליסטיים (ballistic missiles), רחפנים (drones)
- **مصطلحات**: תקיפה תגובתית (retaliatory strike), הסלמה אזורית (regional escalation)

## المعادلة

```
(الطبقة_الأولى) OR (الطبقة_الثانية) OR (الطبقة_الثالثة AND الطبقة_الأولى)
```

## أمثلة

### ✅ سيتم قبول هذه الأخبار:
1. "איראן תוקפת את ישראל" (إيران تهاجم إسرائيل) - الطبقة الأولى
2. "שאגת האריה: ישראל משיקה עמליה" (زئير الأسد: إسرائيل تشن عملية) - الطبقة الثانية
3. "נטנז: מתקן גרעיני איראני" (نطنز: منشأة نووية إيرانية) - الطبقة الأولى + الثالثة

### ❌ سيتم رفض هذه الأخبار:
1. "תיירות באיראן" (السياحة في إيران) - كلمة مستبعدة
2. "ספורט: איראן מנצחת בכדורגל" (الرياضة: إيران تفوز بكرة القدم) - غير ذات صلة
3. "אייר איראן: טיסה חדשה" (إيران إير: رحلة جديدة) - كلمة مستبعدة

## الاستخدام في الكود

```python
from config.keywords import is_relevant_article, get_matching_keywords

# التحقق من صلة المقالة
if is_relevant_article(title, content, language="he"):
    print("✅ المقالة ذات صلة")
    
    # الحصول على الكلمات المطابقة
    keywords = get_matching_keywords(title, content, language="he")
    print(f"الكلمات المطابقة: {keywords}")
```

## تفعيل/تعطيل الفلترة

في `main.py`:
```python
# تفعيل الفلترة (افتراضي)
save_raw_data_to_db(articles, source_id, filter_enabled=True)

# تعطيل الفلترة (حفظ جميع الأخبار)
save_raw_data_to_db(articles, source_id, filter_enabled=False)
```

## إضافة كلمات مفتاحية جديدة

عدّل `config/keywords.py` وأضف الكلمات الجديدة إلى الطبقة المناسبة:

```python
LAYER_1_KEYWORDS = {
    "he": ["كلمة جديدة بالعبرية", ...],
    "en": ["new keyword in English", ...],
    "ar": ["كلمة جديدة بالعربية", ...]
}
```

## ملاحظات

- جميع الكلمات المفتاحية حساسة للحالة (case-insensitive)
- يتم البحث عن الكلمات في العنوان والمحتوى معاً
- يمكن إضافة كلمات مستبعدة في `EXCLUDE_KEYWORDS` لتجنب false positives
